## üöÄ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Modern browser with Web Speech API support

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd medexplain-ai

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Configure your API keys in .env file
# - KIMI_AI_API_KEY: Your Kimi AI API key
# - POLYGON_RPC_URL: Polygon network RPC URL
# - ETHEREUM_RPC_URL: Ethereum network RPC URL

# Start development server
npm run dev
```

### Environment Setup
1. **Kimi AI**: Sign up at [Kimi AI](https://kimi.moonshot.cn/) and get your API key
2. **Blockchain**: Configure RPC URLs for Polygon and Ethereum networks
3. **Voice**: Ensure your browser supports Web Speech API for voice features

### Development Commands
```bash
npm run dev          # Start development server
npm run build        # Build for production
npm run start        # Start production server
npm run lint         # Run ESLint
npm test             # Run tests
```
